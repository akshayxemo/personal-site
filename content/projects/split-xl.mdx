## Overview

SplitXL is an offline-first expense management platform that helps individuals and groups track spending, manage shared expenses, simplify debt settlements, and generate financial reports.

Designed as a Progressive Web App, SplitXL works seamlessly across desktop and mobile devices while storing all user data locally using IndexedDB for privacy and reliability.

## Key Features

### Personal Finance Management

- Track expenses across custom categories
- Monitor budgets and spending trends
- Visualize financial activity through interactive dashboards
- Generate downloadable financial reports

### Group Expense Splitting

- Create groups for trips, events, and shared households
- Support equal, manual, and percentage-based splitting methods
- Manage member balances and settlements
- Track group budgets and spending activity

### Debt Simplification Engine

- Automatically calculate member balances
- Minimize settlement transactions using a greedy optimization algorithm
- Generate simplified payment recommendations
- Track settlement progress over time

### Offline-First Architecture

- IndexedDB persistence through Dexie
- No server dependency required
- Installable as a Progressive Web App
- Local-first privacy-focused design

## Technical Highlights

- Built with React 19 and TypeScript
- IndexedDB data layer with schema migration support
- Zustand-powered state management
- Recharts analytics dashboards
- PDF report generation
- Zod-based validation
- Responsive mobile-first interface

## Challenges Solved

- Designing a local database architecture with migrations
- Implementing efficient debt-settlement algorithms
- Managing complex financial state transitions
- Supporting offline-first user experiences
- Generating exportable reports without backend services
