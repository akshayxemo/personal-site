## Overview

Tea-Leaf is a full-stack e-commerce platform built for online tea retail businesses. The application provides customers with a modern shopping experience while enabling product management, inventory organization, and order processing through a scalable web architecture.

Built using Next.js 14 and TypeScript, the platform focuses on performance, responsive design, and a seamless purchasing workflow.

---

## Key Features

### Product Catalog

* Browse tea products by category
* Detailed product pages
* Product search and filtering
* Responsive product listings

### Shopping Experience

* Add-to-cart functionality
* Cart management
* Dynamic pricing calculations
* Smooth checkout workflow

### User Experience

* Responsive design across devices
* Fast page rendering using Next.js
* Optimized image loading
* Modern UI built with Tailwind CSS

### Data Management

* MongoDB-based product storage
* Dynamic content rendering
* Server-side and client-side data fetching
