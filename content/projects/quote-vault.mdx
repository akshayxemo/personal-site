## Overview

Quote Vault is a cross-platform mobile application built with Flutter that helps users discover, save, organize, and share inspirational quotes.

The application combines cloud-based synchronization, personalized collections, authentication, and modern mobile architecture to provide a smooth and scalable user experience.

## Key Features

### Quote Discovery

- Daily featured quote experience
- Browse quotes by categories
- Search by quote text or author
- Infinite scrolling for content exploration

### Personalization

- Save favorite quotes
- Cloud-synced user collections
- Personalized quote library
- User authentication and account management

### Sharing Experience

- Share quotes with others
- Generate shareable quote cards
- Responsive mobile-first design

### User Experience

- Dark and light themes
- Offline connectivity awareness
- Smooth navigation and state persistence
- Cross-platform support

## Architecture

The project follows Clean Architecture principles with clear separation between:

- Domain Layer
- Data Layer
- Presentation Layer

State management is implemented using Bloc, while dependency injection is handled through GetIt to improve maintainability and scalability.

## Technical Highlights

- Flutter application architecture using Clean Architecture
- Supabase authentication and backend integration
- Bloc-based state management
- Dependency Injection with GetIt
- Functional error handling using Dartz
- Type-safe routing with Go Router
