## Overview

Home Service Agency is a web-based service marketplace that connects customers with service professionals across multiple categories.

Users can browse available service providers, schedule appointments, manage bookings, and track service requests through a centralized platform.

The project was built using Java Servlets, JSP, and MySQL while following a traditional MVC architecture.

---

## Key Features

### Service Discovery

- Browse professionals by service category
- View provider information and service offerings
- Search and filter available services

### Appointment Booking

- Schedule services with preferred date and time
- Manage upcoming bookings
- Track appointment status

### User Management

- Customer registration and authentication
- Profile management
- Booking history tracking

### Administrative Functions

- Service provider management
- Booking oversight
- Platform administration workflows

---

## Technical Highlights

### Backend

- Java Servlet-based application architecture
- MVC design pattern implementation
- Session-based authentication
- CRUD operations for users, bookings, and services

### Database

- MySQL relational database design
- Service, booking, and user management schemas
- Query optimization for booking workflows

### Frontend

- JSP-based dynamic page rendering
- Responsive UI using HTML, CSS, and JavaScript
- Form validation and user interaction workflows
